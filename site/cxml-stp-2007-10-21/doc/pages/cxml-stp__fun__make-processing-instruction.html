Function make-processing-instruction

Lambda List

make-processing-instruction (target data)

Arguments

  • target -- string, an NCName
  • data -- string containing XML characters only

Return Value

an processing-instruction

Details

This function creates a processing instruction.

target must not equal "xml".

data must not contain the substring "?>".
