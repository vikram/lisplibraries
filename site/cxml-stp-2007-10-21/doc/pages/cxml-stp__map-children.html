Package: cxml-stp

Function map-children

Lambda List

map-children (result-type function node)

Arguments

  • result-type -- a sequence type specifier, or nil
  • function -- a designator for a function of one argument
  • node -- a node

Return Value

an sequence of result-type, or nil

Details

Applies function to successive child nodes.

The result-type specifies the type of the resulting sequence. map-children returns nil if result-type is nil. Otherwise it returns a sequence such that element i is the result of applying function to child i of node.
