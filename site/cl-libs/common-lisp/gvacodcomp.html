COMPAGNIES AERIENNES

... Code "IATA"

  • Code > compagnie
  • Compagnie > code

Le "Code IATA" est un système international qui attribue deux lettres (parfois trois ou incluant un chiffre) aux compagnies aériennes (ex. : AF pour Air France). On retrouve ces références sur les billets, les écrans d'information aux aéroports, les guides horaires des compagnies, et aussi parfois les brochures d'agences. Pour des détails sur les compagnies ci-dessous, vous pouvez vous reporter à leurs sites web.
Un même système de codification existe aussi pour les villes et aéroports


. Recherche du nom d'une compagnie d'après son code.
