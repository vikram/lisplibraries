Closure HTML Manual

Example

Parse into LHTML:

(chtml:parse "<p>nada</p>" (chtml:make-lhtml-builder))

=> (:HTML NIL (:HEAD NIL) (:BODY NIL (:P NIL "nada")))

Serialize LHTML back into a string:

(chtml:serialize-lhtml * (chtml:make-string-sink))

=> "<HTML><HEAD></HEAD><BODY><P>nada</P></BODY></HTML>"
