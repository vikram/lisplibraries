Class sax-validation-context-mixin

Superclasses

validation-context, common-lisp:standard-object, sb-pcl::slot-object, common-lisp:t

Documented Subclasses

None

Details

A class that implements validation-context as a mixin for user-defined SAX handler classes.

The mixin will record namespace information automatically, and the user's SAX handler can simply be passed as a validation context to data type functions.
