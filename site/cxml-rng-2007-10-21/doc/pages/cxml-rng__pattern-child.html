Function pattern-child

Lambda List

pattern-child (instance)

Arguments

  • instance -- an instance of pattern

Return Value

an instance of pattern

Details

Returns the pattern's sub-pattern.

(Elements in the full Relax NG syntax allow more than one child pattern, but simplification normalizes the representation so that any such element has exactly one child.)
