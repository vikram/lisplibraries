Package: cxml-types

Function context-find-namespace-binding

Lambda List

context-find-namespace-binding (context prefix)

Arguments

  • context -- an instance of validation-context
  • prefix -- name prefix, a string

Return Value

the namespace URI as a string, or NIL

Details

This function resolves a namespace prefix to a namespace URI in the current context. All currently declared namespaces are taken into account, including those declared directly on the current element.
