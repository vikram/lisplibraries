Function pattern-b

Lambda List

pattern-b (instance)

Arguments

  • instance -- an instance of pattern

Return Value

an instance of pattern

Details

Returns the second of two sub-patterns the pattern instance has.

(Elements in the full Relax NG syntax allow more than two child patterns, but simplification normalizes the representation so that any such element has exactly two children.)
