Function pattern-type

Lambda List

pattern-type (instance)

Arguments

  • instance -- an instance of pattern

Return Value

a cxml-types:data-type

Details

Returns the data type expected at this position.

This type has already been parsed into an object. Its name and the URI of its library can be queried from that object.
