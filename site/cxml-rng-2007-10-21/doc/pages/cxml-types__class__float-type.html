Details

The float data type.

Syntax. A floating-point number in a "scientific notation". C.f. the specification.

Implementation. This type returns a single-float or, on implementations where Infinity and Nan cannot be represented as such, a special symbol that is treated as if it was Infinity or NaN by the built-in ordering.

Parameters. This type is ordered and allows the parameters max-inclusive, min-inclusive, max-exclusive, and min-exclusive.
